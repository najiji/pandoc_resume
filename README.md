The Markdown Resume
===================

**FORK** Fork of <https://github.com/mszep/pandoc_resume> with a more *Markdown-esque* styling.


Instructions:

    git clone https://github.com/najiji/pandoc_resume
    cd pandoc_resume
    vim resume.md   #insert your own resume info
    make

Requirements:

 * ConTeXt
 * pandoc
